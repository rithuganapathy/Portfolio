# Portfolio

This repository contains my personal portfolio website, built to showcase my projects, technical skills, education, and experience in AI, Machine Learning, and Full-Stack Development.

The portfolio highlights some of the work I've been involved in, including multi-agent AI systems, research automation tools, and machine learning projects.

## About Me

I'm Rithu Ganapathy, currently pursuing an M.Tech in Artificial Intelligence & Data Science.

My interests lie at the intersection of software engineering and AI. I enjoy building systems that can reason, automate tasks, and solve real-world problems. Alongside AI and machine learning, I also work with modern web technologies and full-stack development.

## Featured Projects

### ARPA

An autonomous research platform that transforms research papers into structured information and runnable machine learning experiment code using multi-stage document processing and LLM-based reasoning.

### ArenaGen

A multi-agent framework capable of generating reinforcement learning environments directly from natural language descriptions. The system uses multiple specialized agents to design, validate, and refine generated environments.

### Distributed Gradient Descent with Hybrid PSO

My M.Tech research project focused on improving distributed neural network training through a hybrid approach that combines Gradient Descent and Particle Swarm Optimization while reducing communication overhead.

## Running Locally

Clone the repository:

```bash
git clone https://github.com/rithuganapathy/Portfolio.git
```

Navigate to the project directory:

```bash
cd Portfolio
```

Open `index.html` in your browser or run it using VS Code Live Server.
